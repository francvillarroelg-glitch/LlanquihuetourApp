# Llanquihue Tour

## Descripción
Aplicación Java que lee información de tours desde un archivo de texto,
crea objetos Tour y los almacena en un ArrayList.

## Estructura

src/
├── model/
├── data/
└── ui/

resources/
└── tours.txt

## Ejecución

1. Abrir el proyecto en IntelliJ IDEA.
2. Ejecutar la clase Main.
3. El sistema cargará los tours desde el archivo.
4. Mostrará todos los tours y los filtrados.
